Colors

Please find below our color palette.

#222222

Objects are never ever 100% black (#000). The default text color is #222. We call it $mine-shaft-black.

#e2f4fc

This light blue is used as a general background color for e.g. in menus and panels. We call it $foam-blue. The related hover color and selected color is $tropical-blue.

#2196d8

This medium blue is used for several elements: link color, logo color and default button color. We call it $curious-blue.

#66bb6a

This green is used for buttons with primary actions. We call it $fern-green.

#ea4335

This red is what we use for alerts/notifiers where the user should take action. Some buttons will also have this color (read more in buttons). We call it $cinnabar-red.

#f4842e

This pretty orange is only used for alerts/notifiers with second priority, e.g. a label saying "New". We call it $jaffa-orange.

#fff9c4

This post-it light yellow is used for background color in information panels. We call it $lemon-chiffon-yellow. The accent color is called $paris-daisy-yellow.

Buttons

All actions are not equal. By using visual distinction, we are helping the users to make the right choices. The visual presentation of actions should match their importance.

Markup

Call the button element with <a class="button"></a>.
Then select the type of button, and add as an additional class: <a class="button button--cta"></a>

CTA (green) buttons

The button--cta button. Used for important actions built to support our business model.
A cta ("call to action") is a clear direction that encourages the visitor to take a certain action.

Button sizes

The default size is a large button. We also offer medium and small.
So, if there is a need to adjust the button size, add as an additional class: <a class="button button--cta button--medium"></a>

Please note:

For buttons on mobile devices there is sometimes a need for a button to be wide, which is why we have included have an option for full device width. Simply call button-full.

Buttons with icons

Do not use icons inside of buttons. Instead, make an effort to write clear and unambiguous copy.

Because: nothing says ”edit” like ”edit”. In the battle of clarity between icons and labels, labels always win. The best icon is a text label.

Disabled buttons

Disable buttons by adding class disabled, or by adding HTML attribute disabled.

Class Naming Convention

Write lower case, using dashes.

Correct: 

.some-class

Incorrect: 

.someClass
.some_class
.someclass
.SomeClass

Declarations

  • Place Scss/css declarations in alphabetical order. This is to easily find what you are looking for, and to eliminate the risk of duplication.
  • Use variables as much as possible.
  • Wrap inside of so called ”egyptian brackets”.
  • Indent by 4 spaces.
  • Arrange pseudo classes like below.

As shown in this example: 

.some-class {
    color: $color;
    font-family: $font;
    padding: 1rem;
    width: (100% / 3);
    &:hover {
        color: darken($color, 5%);
    }
}

Nesting

Nesting is an expensive operation. Always try to find a direct selector. The class name .some-list__item is always to prefer instead of: .some-list > div + ul li. Read more in the BEM section.

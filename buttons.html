Buttons

All actions are not equal. By using visual distinction, we are helping the users to make the right choices. The visual presentation of actions should match their importance.

Button sizes

The default size is a large button. We also offer medium and small.
So, if there is a need to adjust the button size, add as an additional class: class="button medium"

Please note:

Sometimes there is a need for a button to be wide, which is why we have included have an option for full device width. Simply call class="button expand".

Buttons with icons

Do not use icons inside of buttons. Instead, make an effort to write clear and unambiguous copy.

Because: nothing says ”edit” like the word ”edit”. In the battle of clarity between icons and labels, labels always win. The best icon is a text label.

Disabled buttons

Disable buttons by adding class disabled, or by adding HTML attribute disabled.

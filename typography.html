Typography

The main font used is Droid Sans, served via Google Fonts. The Droid family of fonts is designed by Steve Matteson. The Droid fonts provide optimal quality and reading comfort on a mobile handset. They are optimized for use in application menus, web browsers and for other screen text.

Font Family

The font contains a regular weight (400) and a bold weight (700). Droid Sans does not offer a weight for italics. When using <i> markup for italics, the regular font will be mathematically skewed and quite unaesthetic. For this reason we use Droid Serif for italics. This feature is built into the <i> tag in typography.scss.

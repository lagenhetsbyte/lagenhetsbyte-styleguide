Buttons

All actions are not equal. By using visual distinction, we are helping the users to make the right choices. The visual presentation of actions should match their importance.

Markup

Call the button element with <a class="button"></a>.
Then add the type of button as an additional class: button--blue.
The default size is a large button.
However, if there is a need to adjust the button size, add class button--medium or button--small.

Green buttons

The cta button. Used for important actions related to our business model.

Please note:

Do not use icons inside of buttons. Instead, make an effort to write clear and unambiguous copy.

Because: nothing says ”edit” like ”edit”. In the battle of clarity between icons and labels, labels always win. The best icon is a text label.

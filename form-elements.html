Form elements

We have decided to not use the standard form elements. Instead, we use our own elements by applying new base styles on the form elements themselves, or the label::before and label::after pseudo elements.

The checkbox

This is the correct markup for the checkbox: 

<input id="x" type="checkbox" />
<label for="x">Checkbox label</label>

The radio button

This is the correct markup for the radio button: 

<input id="x" type="radio" />
<label for="x">Radio button label</label>

The select

This is the correct markup for the select: 

<select>
    <option>Option 1</option>
    <option>Option 2</option>
</select>

The textarea

This is the correct markup for the textarea: 

<textarea>
    Textarea
</textarea>

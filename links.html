Button or link?

Link

When you are performing actions which lead to browsing of content, viewing pages, toggling hidden/visible partial content — use a link.

We have two types of links:

  • The regular link, which is blue and has no underline. Example
  • The subtle link, which is grey has underline. Example

Button

When you are performing actions which affect the backend, e.g. creating an account, signing in, making a purchase, posting an article, sending a message — use a button.

Read about our buttons

So, in short:

Going to the oven: use a link. Starting the oven: use a button.
